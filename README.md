# DAVE3606 Ressurseffektive programmer & DATS2300 Algoritmer og datastrukturer – Exam Prep


This repository contains my personal exam preparation materials, mainly for **DAVE3606 Ressurseffektive programmer**. However, much of the content and practice is highly relevant to **DATS2300 Algoritmer og datastrukturer**, as the subjects overlap in their focus on algorithms, data structures, and resource-efficient programming.

## About the Subjects

### DAVE3606 Ressurseffektive programmer
Focuses on resource-efficient programming, including:
- Understanding what makes programs computationally or memory intensive
- Generic programming concepts
- Design patterns and their practical examples
- Using resource-efficient languages and methods
- Creating custom memory-efficient data structures
- Applying design patterns, generic programming, and abstractions
- General competence in developing resource-efficient programs and techniques for optimization at various abstraction levels

### DATS2300 Algoritmer og datastrukturer
Covers:
- Data structures: tables, lists, stacks, queues, heaps, hash tables, trees, graphs, files
- Algorithms: counting, insertion, searching, deletion, traversal, sorting, optimization, compression
- Analysis, design, and implementation of data structures and algorithms
- Choosing appropriate structures and algorithms for specific problems
- Using both custom and standardized solutions for complex problems
- Competence in discussing and advising on the use of data structures and algorithms

## About This Repository

This repo is **not** a complete reflection of all coursework or exam content. It is a collection of code and notes created mainly as a way to learn and practice Python before the exam. The focus is on implementing and testing classic algorithms and data structures, especially those relevant for resource efficiency and algorithmic understanding.

### Structure
- `Algorithms/` – Implementations and tests for sorting algorithms (Bubble, Bucket, Counting, Heap, Insertion, Merge, Quick, Radix, Selection)
- `DFS (Depth First Search)/` – Depth First Search algorithm and tests
- `HashMap/Python(Phonebook)/` – Simple phonebook application using hash maps, with GUI and tests

### Purpose
- Practice Python coding for exam readiness
- Reinforce understanding of algorithms and data structures
- Explore resource-efficient programming techniques
